📌 TOPIC: R + RStudio — Complete Setup + Interface

WHAT IS R?
R = Programming language for
    statistical computing and graphics

Created in: 1993
By: Ross Ihaka and Robert Gentleman
    (University of Auckland, New Zealand)

R vs RStudio — VERY IMPORTANT DIFFERENCE
R = The ENGINE                                
Think of R like a CAR ENGINE.                
It does all the actual work.                 
But driving an engine alone is difficult.

RStudio = The CAR (with steering, seats)     
RStudio is the comfortable interface         
that makes using R engine easy!             
It needs R installed first to work.  

HOW TO DOWNLOAD R
STEP 1:
Open any browser
Go to: cran.r-project.org

STEP 2:
You see an option:
→ Download R for Windows   ← Click this (Windows users)

STEP 3 (Windows):
Click "base"
Click "Download R x.x.x for Windows"
File downloads (about 80 MB)

STEP 4:
Open downloaded file
Click "Yes" when asked permission
Click "Next" on all screens
Keep all default settings
Click "Finish"

✅ R is installed!

HOW TO DOWNLOAD RSTUDIO
STEP 1:
Go to: posit.co/download/rstudio-desktop

STEP 2:
You see "Download RStudio Desktop"
Click the FREE version button
(It says "RStudio Desktop - Free")

STEP 3:
File downloads (about 200 MB)

STEP 4:
Open downloaded file
Click "Yes" when asked permission
Click "Next" on all screens
Click "Install"
Click "Finish"
✅ RStudio is installed!

RSTUDIO INTERFACE — 4 PANELS
This is the MOST important thing to understand today!

When you open RStudio, you see 4 panels:

─────────────────────┬─────────────────────┐
│                     │                     │
│   TOP LEFT          │   TOP RIGHT         │
│                     │                     │
│   SCRIPT EDITOR     │   ENVIRONMENT       │
│   (Write code here) │   (See variables)   │
│                     │                     │
├─────────────────────┼─────────────────────┤
│                     │                     │
│   BOTTOM LEFT       │   BOTTOM RIGHT      │
│                     │                     │
│   CONSOLE           │   FILES/PLOTS/HELP  │
│   (Run code here)   │   (See outputs)     │
│                     │                     │
└─────────────────────┴─────────────────────┘
PANEL 1: SCRIPT EDITOR (Top Left)
→ Type your R code here
→ Press Ctrl + Enter to run ONE line
→ Press Ctrl + Shift + Enter to run ALL
PANEL 2: CONSOLE (Bottom Left)
→ Type R command after >
→ Press Enter
→ See result immediately
PANEL 3: ENVIRONMENT (Top Right)
→ See all your variables
→ Check dataset size
→ Know what is in memory
PANEL 4: FILES/PLOTS/HELP (Bottom Right)

This panel has MULTIPLE TABS:

TAB 1: FILES
→ See files on your computer
→ Navigate folders
→ Open R scripts
→ Like Windows File Explorer

TAB 2: PLOTS
→ Graphs appear here
→ When you create a chart
→ It shows here automatically
→ Can save/export graphs

TAB 3: PACKAGES
→ R add-on libraries
→ Install new packages
→ Load existing packages
→ Like SAS procedures (built-in tools)

TAB 4: HELP
→ Search R documentation
→ How to use functions
→ Examples for every command

TAB 5: VIEWER
→ Preview HTML outputs
→ Interactive documents
→ R Markdown previews
